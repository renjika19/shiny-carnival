Directives!

A directive is a way that we can invoke Angular functionality within an html file. They act as syntax that can be used in the html format without breaking the front end.

*ngIf

What does the * mean in front of ngIf?

Any directive that has an asterisk(*) means that it is a 'structural directive.' This means that the directive in question will actually manipulate the html directly. That is, it will be capable of physically deleteing or adding html on the spot for us

Attribute Directive

Attribute directives are directives that are applied as attributes in the elements themselves, like structural directives. However thes directives do NOT aim to change the actual structure of the html, but rather change the style.
