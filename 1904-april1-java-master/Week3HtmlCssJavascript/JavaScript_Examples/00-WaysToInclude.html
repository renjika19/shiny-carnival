My First JavaScript

  • JavaScript is the programming language of HTML and the Web
  • It is used to program the behavior of web pages
  • In HTML, JavaScript code must be inserted between the script tags
  • You can place any number of scripts in an HTML document
  • Scripts can be placed in the body or in the head section or both
  • JavaScript can be used to change html content
  • It can also be used to change attributes and CSS
  • Can hide/show html elements
  • Make webpages more dynamic
  • Java =/= JavaScript
