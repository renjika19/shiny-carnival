DOM Manipulation

DOM stands for Document Object Model. It is a representation of all of the elements on a webpage. You can think of it as a family tree that contains all of the document's html elements and details regarding each element's lineage



There are various JavaScript functions that you can use in order to affect the DOM. This is called "DOM Manipulation"



There are different ways to get elments:
